# 猜拳对战！(Rock-Paper-Scissors Battle)

一款基于 Godot 4 引擎的本地单机回合制猜拳对战游戏，灵感来源于石头剪刀布 + 三国杀座位距离机制。

## 游戏概述

玩家选择角色后与 AI 进行猜拳对决。每回合所有存活玩家出拳（石头/剪刀/布），胜者获得一次行动机会——可以选择**聚气**（积累能量）或**使用技能**攻击敌人。最后存活的玩家获胜。

## 核心机制

### 猜拳规则
- 石头 > 剪刀，剪刀 > 布，布 > 石头
- 多人获胜时进入**加赛**：胜者之间重新猜拳，直到唯一胜者产生
- 平局（全部出相同或三种手势都有）→ 重新出拳
- **麻痹**状态下自动跳过出拳

### 距离系统
- 玩家排成**环形**（座位制），距离 = min(顺时针, 逆时针) + 偏移修正
- 技能有 `min_range` / `max_range` 限制，超出射程不可使用
- 距离可通过技能效果改变（拉近/拉远）
- 死亡玩家从环形中移除

### 技能效果系统（8 种）
| 类型 | 说明 |
|------|------|
| DAMAGE | 直接伤害，遵循影分身 → 无限盾 → 数值盾 → HP 的吸收顺序 |
| SHIELD | 护盾：-1 = 一次性全挡，>0 = 数值护盾 |
| PARALYZE | 麻痹：跳过指定回合的出拳 |
| CHANGE_DISTANCE | 改变距离偏移 |
| HEAL | 恢复生命值（不超过最大 HP） |
| DELAYED_DAMAGE | 延迟伤害：数回合后触发 |
| CLONE_SHIELD | 影分身：抵挡一次伤害 + 充能加成 |
| UNLOCK_SKILL | 永久解锁新技能 |

### 伤害吸收顺序
```
影分身（一层挡一次）→ 无限盾（-1 一次性）→ 数值盾（>0）→ HP
```

## 游戏流程

```
主菜单 → 角色选择 → 战斗场景 → 结算画面
  ↑         ↑           ↑           |
  └─────────┴─── 返回 ───┴── 菜单 ───┘
```

1. **主菜单**：选择 PvE 对战、设置、退出；角色卡展示
2. **角色选择**：选择你的角色，配置 AI 数量（1-7），开始游戏
3. **战斗场景**：出拳 → 结算 → 加赛 → 行动 → 执行 → 淘汰 → 下一回合
4. **结算画面**：胜负展示 + 淘汰顺序 + 统计/回放/技能记录三标签分析

## 技术栈

- **引擎**：Godot 4.6+ (GDScript)
- **物理**：Jolt Physics 3D
- **渲染**：Direct3D 12 (Windows)
- **分辨率**：1280×720 (设计分辨率 960×540，canvas_items 拉伸)

## 项目结构

```
├── core/                   # 核心游戏逻辑
│   ├── game_manager.gd     # 游戏主循环状态机（自动加载）
│   ├── scene_manager.gd    # 场景导航与数据传递（自动加载）
│   ├── settings_manager.gd # 设置持久化（自动加载）
│   ├── player_state.gd     # 玩家运行时状态
│   ├── round_resolver.gd   # 手势结算与技能效果（纯静态）
│   ├── ai_controller.gd    # AI决策逻辑
│   ├── distance_system.gd  # 环形座位距离系统
│   ├── match_record.gd     # 对局数据容器
│   ├── player_match_stats.gd  # 玩家统计数据
│   └── ...                 # 快照/日志数据类
├── resources/              # 数据定义与角色资源
│   ├── character_data.gd   # 角色数据 Resource
│   ├── skill_data.gd       # 技能数据 Resource
│   ├── skill_effect.gd     # 技能效果 Resource
│   └── characters/         # .tres 角色定义 + 头像
├── scenes/                 # 场景文件
│   ├── main_menu.tscn/gd   # 主菜单
│   ├── character_select.tscn/gd  # 角色选择
│   ├── main.tscn/gd        # 战斗入口
│   ├── game_ui.tscn        # 战斗 UI 组件
│   ├── game_over.tscn/gd   # 结算画面
│   └── settings.tscn/gd    # 设置面板
├── ui/
│   └── game_ui.gd          # 战斗 UI 完整逻辑（1500+ 行）
├── assets/ui/              # UI 贴图资源
├── project.godot           # Godot 项目配置
└── export_presets.cfg      # 导出预设
```

## 运行方式

1. 安装 [Godot 4.6+](https://godotengine.org/)
2. 克隆项目后用 Godot 打开 `project.godot`
3. 按 F5 运行（或点击"运行项目"）

导出 Windows 可执行文件：
- 项目 → 导出 → Windows Desktop → 导出项目

## 设置选项

| 设置 | 默认值 | 说明 |

| 出拳倒计时 | 60s | 30s/60s/90s/无限/自定义 |

| 动画速度 | 1.0× | 0.1× - 10.0× |
| 全屏模式 | 关 | 切换全屏/窗口 |
